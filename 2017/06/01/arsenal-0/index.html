足总杯夺冠，官宣教授续约的晚上

发表于 2017-06-01 | | 人阅读

不平凡的一天

今天终于官宣了温格教授续约。悬着的心终于放下了。睡醒觉收到了官方会员发过来的中国行的付款通知，迟迟不肯付款。因为如果教授不续约，似乎北京的吸引力就没那么大了。今天一直不停的刷微博，等着今天的大新闻。终于在快下班的时候看到社长的微博，说教授已经在合同上签字。晚上回家看到了官宣的消息。激动的心情难以言表，于是马上付了中国行的死忠区的款。从上周六足总杯问鼎，到今天教授续约，对我来说，是接二连三的好消息。现在就盼着夏季转会引援给力，374和272可以留下，让教授在剩下的任期内，有机会问鼎英超和欧冠。简直完美。如果真有那么一天，我一定会去现场看那场比赛，不管我身在何处，身居何职，我想我都会去。

就这样爱上阿森纳

回想一下，我是如何从一个对足球一无所知，不知道场上都有什么位置，说不出任何一个球员的名字，不知道现在火的球队的名字，成为一个几乎每场比赛都会看，对厂子的球员如数家珍，看得出越位，叫得出英超20只球队的名字，认得出几大俱乐部的球星，甚至是大致的排位都心中有数的球迷呢？

这要追溯到2016年2月14日，阿森纳和当时排名榜首的莱斯特城的那场比赛。因为身边有朋友从小就喜欢阿森纳，久了也耳濡目染，但是第一次完完整整的看的一场比赛。便是这场，那时候不懂怎么找信号源，免得麻烦，还是付费看的，最后维尔贝克读秒绝杀，太刺激了！苦苦等待了90多分钟，似乎就为了这一刻的绝杀。第一次深深的感受到了足球的魅力。于是开始深入的了解这家俱乐部，才知道温格教授已经执教阿森纳20年之久，16–17赛季，是他执教枪手的第21个赛季。很难想象，是怎样的一种钟情和专一，让一个教练为一个球队献出20多年的时光，看教授之前的视频，刚来枪手执教之时还是一头黑发，到现在的满头白发。不禁由衷的钦佩和难以名状的喜爱。似乎正如自己对待喜欢的种种，从那时起，便成为一名温蜜。

16–17赛季，几乎一场不落的看完了厂子38轮比赛，记得赛季初连续十多场不败，第一次在电影院看对战曼联的比赛，吉鲁最后关头替补上场，逼平曼联。再到后来12月份的时候连续2：1被埃弗顿和曼城逆转，欧冠赛场被拜仁踢了总分10：2。到二三月份各种温格out的言论，心疼教授。后来又0：3输给水晶宫，2：0输给同城死敌热刺。输给热刺后，让教授下课的声音更是愈演愈烈，不过也有派飞机拉横幅支持教授留任的枪迷。好在5月，枪手没有让大家失望，只可惜发力太晚了。5月份全胜战绩结束英超16–17赛季。赢了曼联、埃弗顿。在上周六的晚上赢了英超冠军切尔西，夺得足总杯冠军，叫保温杯，似乎也不为过，足总杯夺冠4天后，教授宣布续约两年。

今年夏天，北京见

想来去年底让在英国的小妹妹帮自己买了本温格20周年纪念画册和别人写的温格的传记。自己也买了中文版教授的传记来看。教授执教阿森纳的21年，就是阿森纳近代的活历史。为了亲眼见到温格，今年夏天一定要去北京。其实有时候，有些事情很神奇。有些东西你从来不会觉得自己会如此喜欢。有的人你从来都不会觉得你会爱上，但当你发现时，却已经很难说放弃。有些爱好从来不觉得自己会坚持很久，可随着时间的推移，竟然发现已经无形之中成了生活的一部分。像骑车，像阿森纳。

每当看到白发苍苍的教授发表言论，临场指挥，进了球开心的笑，输了球抱头的样子，就让人不得不尊敬，真的好想多看几年教授执教的破厂。今年是教授执教史上第一次跌出前四，只希望明年，17–18赛季，破厂和教授一起交出一份对彼此满意的答卷。

希望不久的将来，再次看教授捧杯。
